# Labirinto
## Il progetto unity deve contenere i seguenti elementi di base:
- Un terreno di sfondo
- Un character controller
- Un labirinto generato in modo procedurale tramite una funzione (possibilmente ricorsiva) 
- Delle testure per gli elementi principali del progetto
 ## Verrà anche valutato in modo positivo:
- La presenza di più livelli
- Delle schermate di Vittoria e Sconfitta
- Un orologio o cronometro della partita 
- Un sistema di punti tramite token presenti sulla mappa da raccogliere
- La grafica applicata all'ambiente esterno al labirinto
- Una tematica del gioco
- Altre vostre idee per migliorare il gioco in se
